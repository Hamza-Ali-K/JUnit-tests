# JUnit-tests
This repository contains Java programs and JUnit test cases created for Software Quality Engineering lab tasks. The project demonstrates how unit testing is performed in Java using JUnit to verify the functionality of different classes and methods.

# Lab Tasks Included
Triangle Type Checker (Equilateral, Isosceles, Scalene, Invalid)
Calculator Operations
Addition
Subtraction
Multiplication
Square
Cube
Number Utility Functions
Palindrome Check
Factorial
Sum of Digits
Prime Number Checker
Even/Odd Number Checker
JUnit Test Classes
Test Suites
Maven pom.xml Configuration
# Technologies Used
Java
JUnit
Maven
IntelliJ IDEA / Eclipse

# How to Run the Project
Clone the repository:
git clone https://github.com/Hamza-Ali-K/JUnit-tests.git
Open the project in IntelliJ IDEA or Eclipse.
Make sure Maven dependencies are installed.
Run the test classes or test suite using JUnit.
# How to Run Using Maven

Open terminal in the project folder and run:

mvn test
# Purpose of This Project

The purpose of this project is to understand unit testing in Java and learn how JUnit test cases are used to verify program functionality and improve software quality.
